# Linux Apache Web Server Portfolio

This project demonstrates a **working Apache web server** on Linux with a **portfolio-ready page**.

---

## Project Files

- `index.html` → Custom HTML page showing Apache status and GitHub link  
- `screenshots/` → Screenshots of Apache install, service status, browser page, and IP config

---

## How to View

1. Open `index.html` in a browser  
   or run Apache on Linux and go to `http://localhost`  

2. Screenshots show the installation and verification steps.
